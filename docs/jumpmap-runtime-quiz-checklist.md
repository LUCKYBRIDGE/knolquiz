# 점프맵-퀴즈코어 통합 체크리스트

---

## 1) 준비
1. 저장 데이터 백업(권장)
- `mkdir -p backups && cp -f save_map/*.json backups/` (필요한 파일만)
2. 로컬 서버 실행(반드시 repo 루트에서)
- `cd /Users/baekjiyun/Desktop/WAN/math-net-master-quiz`
- `node scripts/jumpmap-local-serve.mjs`
3. URL 확인
- `http://127.0.0.1:5173/jumpmap-editor/`
4. 콘솔 에러 없음 확인
5. `node --check` 확인
- `node --check public/jumpmap-editor/editor.js`
- `node --check public/jumpmap-editor/test-runtime.js`
- `node --check public/jumpmap-editor/map-io-utils.js`

Expected:
- 초기 진입 시 렌더/스크립트 에러 없음

## 2) 기본 모드 확인
1. 게임 모드 기본값이 `점프맵`인지 확인
2. 공통 모드 설정 시 전 플레이어가 같은 모드로 시작하는지 확인
3. 모드 어댑터가 `jumpmap` 규칙셋을 로드하는지 확인

Expected:
- 멀티플레이 기본값이 공통 모드로 적용됨

## 3) 게이지 규칙 확인
1. 지상 이동 시 게이지 감소 확인
2. 지상 점프 시 게이지 감소 확인
3. 게이지 0 상태에서 지상 점프 차단 확인
4. 공중에서 이동 입력 허용 확인
5. 공중 더블점프 차단 조건 확인
6. 퀴즈 정답 시 게이지 큰 회복 확인
7. 퀴즈 오답 시 게이지 `+0` 확인
8. 퀴즈 오답 결과 후 `3초 딜레이` 확인

Expected:
- 지상 차단/공중 허용 규칙이 분리되어 일관 동작

## 4) 퀴즈 루프 확인
1. 퀴즈 버튼 클릭 -> 문제 진입
2. 문제 1개 풀이(정답/오답 각각)
3. 문제 결과(정답/오답)가 명확히 표시되는지 확인
4. 각 문제 종료 후 선택 팝업(`다음 문제` / `맵으로 복귀`) 표시 확인
5. `다음 문제` 동작 확인
6. `맵으로 복귀` 동작 확인

Expected:
- 퀴즈 결과에 따라 게이지 반영
- 복귀 후 조작/물리 정상

## 5) 멀티플레이 분리 확인 (2/3/4/5/6인)
1. 플레이어 A만 퀴즈 진입
2. 플레이어 B~N은 이동/점프 지속
3. 각 플레이어 게이지가 독립 반영되는지 확인
4. 플레이어명 라벨 표시 확인
5. 각 플레이어 입력이 자기 구역 안에서만 동작하는지 확인
6. 플레이어 A의 퀴즈 팝업이 B~N 화면에 나타나지 않는지 확인

Expected:
- `playerId` 단위 상태 오염 없음
