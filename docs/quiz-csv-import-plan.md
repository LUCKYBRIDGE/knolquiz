# Quiz Core 확장 계획: CSV 업로드 기반 문제 생성

작성일: 2026-02-28  
상태: 2차 반영 (런처 업로드 + 선택지1~4 + 주관식 채점 규칙)

## 목표
- 교사가 CSV 파일 업로드만으로 문제를 즉시 생성/출제할 수 있게 한다.
- 기존 이미지 기반 전개도 문제/영역모델 구조화 문제와 공존한다.
- 문제별 시간 제한(초)을 선택적으로 적용한다.

## CSV 사양 (현재)
- 기본 헤더: `문제 내용`, `선택지1`, `선택지2`, `선택지3(선택)`, `선택지4(선택)`, `정답번호`
- 선택 헤더: `문제시간(초)`, `정답인정단어`, `단어포함시정답처리여부`
- 정답번호: 1부터 시작하는 선택지 인덱스
- `정답인정단어`가 채워진 행은 주관식으로 처리하며 선택지는 무시
- `단어포함시정답처리여부`: `Y`면 포함 일치, `N`(기본)이면 완전 일치

## 우선순위
1. CSV 파서 + 스키마 검증 연동
2. 런처 첫 화면 업로드/템플릿 다운로드
3. 텍스트 객관식 + 주관식 렌더링 경로
4. 문제별 시간 제한(기존 전역 시간 제한과 병행)
5. 이후 확장: CSV에서 구조화 문제(PVAM)까지 매핑
