퀴즈 게임 시작

인원, 퀴즈, 게임을 먼저 고르고 바로 시작합니다. 점프맵 선택 시 캐릭터도 고를 수 있습니다(현재 2종).

게임 시작 설정

1. 플레이 인원
2. 퀴즈 선택
검색어와 필터를 함께 사용해 원하는 퀴즈를 빠르게 찾을 수 있습니다.
2-1. 문제 양식/업로드
CSV/JSON 업로드 파일을 로컬 저장목록(최대 20개)에 보관
math-net 문제은행 퀴즈 앱에서 저장한 문제팩(JSON)도 업로드할 수 있습니다.
권장 흐름: math-net에서 문제팩(JSON) 다운로드 → 현재 기기에서 이곳에 업로드 → 퀴즈 선택 후 시작
현재 업로드된 문제 파일 없음
업로드된 문제 미리보기가 여기에 표시됩니다.
저장된 업로드 파일이 없습니다.
2-2. 종료/진행 설정 (퀴즈+점프맵)
종료 기준
기본 퀴즈 모드에서 적용됩니다.
현재 선택한 게임 모드 기준으로 위 설정이 적용됩니다.
3. 게임 선택
4. 캐릭터 선택 (점프맵)

플레이어별 캐릭터는 아래 학생 번호 행에서 개별 변경할 수 있고, 해당 번호의 대표 캐릭터로 저장됩니다.

운영 맵에서 시작 위치 목록을 불러오는 중입니다.

시작 요약

맵 에디터 (개발도구) 기존 문제은행 퀴즈 앱 (개발도구) 업로드 문제 생성 퀴즈 (신규) 학급관리 (로컬) 명예의 전당
현재 점프맵 런타임 분리 작업 중입니다. 사용자 플레이는 운영 경로를 우선 사용하고, 일부 경계 구간은 레거시 호환 모드로 연결됩니다.
